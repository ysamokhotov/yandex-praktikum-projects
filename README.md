## Основная информация :bookmark_tabs:
Данный репозиторий содержит оформленные проекты в ходе обучения на курсе по аналитике данных от Яндекс.Практикум.
	
## Инструментарий :hammer:

* numpy v1.23
* matplotlib v3.6.0
